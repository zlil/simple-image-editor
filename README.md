# Simple-Image-Editor
> A Vue.js & Node.js project

## Explanation

* Paste Image Url
* Choose Text Color
* Add Your Text
* Save & Enjoy

``` bash
Build Setup Front End

# install dependencies
cd frontend
npm install

# serve with hot reload at localhost:8080
npm run dev

Build Setup Back End

# install dependencies
cd backend
npm install

# run server localhost:3000
node server.js
```
